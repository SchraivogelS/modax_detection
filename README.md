## Robust cochlear modiolar axis detection
Based on a matlab implementation by Prof. Wilhelm Wimmer and ported to Python by Dr. Noelle Harte and Dr. Stephan Schraivogel.

This work is published as:
Wimmer W, Vandersteen C, Guevara N, Caversaccio M, Delingette H. Robust cochlear modiolar axis detection in CT. In International Conference on Medical Image Computing and 
Computer-Assisted Intervention 2019 Oct 10 (pp. 3-10). Cham: Springer International Publishing. [doi](https://doi.org/10.1007/978-3-030-32254-0_1)

There is an extended version for second order kinematic surface fitting, which is currently not implemented:
Wimmer W, Delingette H. Second order kinematic surface fitting in anatomical structures. Medical Image Analysis. 2025 Apr 1;101:103488. [doi](https://doi.org/10.1016/j.media.2025.103488)
